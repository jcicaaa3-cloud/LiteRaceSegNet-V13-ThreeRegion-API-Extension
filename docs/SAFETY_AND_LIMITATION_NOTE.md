# Safety and Limitation Note

LiteRaceSegNet V13 API output is intended for academic, capstone, portfolio, and prototype review.

The output is **not** a certified road safety diagnosis and should not be used alone for:

- official road maintenance priority,
- legal judgment,
- insurance evaluation,
- administrative decision-making,
- public safety certification.

Reported V13 values such as mIoU, Damage IoU, parameter count, and CUDA latency should be treated as pilot-scale evidence under the recorded evaluation setting, not production validation.
